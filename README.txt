[Program 5]
[Description]
This program creates random passwords based on the frequency of how a letter follows another in a word, from a given file (Example: “True”, “r” follows “T”).
To run it use 'java Passwords reference-Filename N k', where reference-filename is the file you want to base the letter frequencies from, N the number of passwords to produce, and k the length of the passwords.
There is only one file that does everything in the main method, with helper methods that do non-essential things. The main function is creating the passwords, which is done by taking the sum of the number of times the current letter has appeared behind each letter of the alphabet inside the file and creating a random number that falls between 0 and the sum. Each letter will have varying ranges and the next letter in the password is determined by where the random number falls.
Everything prints to the terminal but the table is very large so make sure to enlarge the terminal to max capacity.
